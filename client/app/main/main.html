API Basejump: Timestamp microservice

User stories:
  1. I can pass a stirng as a parameter, and it wil check to see whether that string contains either a unix timestamp of a natural language date (example: January 1, 20167)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties

Example usage:

https://kinley-timestamp.herokuapp.com/December%2015,%202015
https://kinley-timestamp.herokuapp.com/1450137600

Example output

{"unix": 1450137600,"natural":"December 15, 2015"}
